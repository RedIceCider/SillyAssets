# SillyAssets

Adds alternate avatars for alternate greetings and adds the `{{asset::asset name}}` macro. Asset data is stored on card so they can be exported as well.

Install extension > bottom magic wand > Manage extensions > Add URL or upload image (stored inside the card extension's field as base64) > save

Use `/chat-avatar {url}` to change avatar for current chat only.
Use `/user-avatar {url}` to change user's avatar for current chat only.

It should just work, but just in case, do make a backup of your card before uploading an image. No warranties provided.
